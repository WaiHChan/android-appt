This is a README.txt

This is project 5. An Android version of appointment application.

There is a "Help" button to lead to print "README.txt".

There is a "Display All" button to display all the appointment created.
If there are any appointment books, it will display the owner of the book.
The user can click on the owner's name, and it will show the owner's appointments.

There is a "Search By Name" button to search for owner.

There will be a "Search By Name And Time" button to search for a owner appointments in particular time.

There will be a "Make Appointment" button to create an appointment.
